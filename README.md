# superAr

O projeto SuperAr possui dois submódulos sendo eles: superAr_frontend e o superAr_backend.

Portanto, para clonar o projeto trazendo os submódulos é necessário executar o comando: git clone --recurse-submodules

Por causa dos submódulos existem 3 repositórios: superar, superar_frontend e superar_backend. O desenvolvimento está restrito aos repositórios superar_frontend e superar_backend. 

Como boa prática de projeto os desenvolvedores deverão fazer commits, push e merges para o branch master somente ao ter versões funcionais dos códigos. Enquanto o código estiver em desenvolvimento ou testes, os desenvolvedores deverão utilizar o branch desenvolvimento ou outros branchs.
